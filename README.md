# E-Commerce QA Test Suite

This is a beginner-friendly manual testing portfolio project for an e-commerce demo website.

## Site Under Test

- Website: SauceDemo
- URL: https://www.saucedemo.com/
- Application type: E-commerce demo app

## Project Goal

Learn and practice:

- Test case writing
- Bug reporting
- QA thinking
- SDLC and STLC basics
- Functional, smoke, and regression testing
- Defect lifecycle and test traceability

## Features Covered

- Login
- Signup
- Product search
- Product filters and sorting
- Cart
- Checkout

Note: SauceDemo does not currently provide a real signup feature or free-text product search. These are included as expected e-commerce requirements so you can practice identifying requirement gaps, blocked tests, and defect reporting.

## How To Use This Project

1. Read the test plan.
2. Open SauceDemo in a browser.
3. Execute the test cases one by one.
4. Update the actual result and status in the workbook.
5. Log defects in the bug report sheet.
6. Use the RTM sheet to confirm every requirement has test coverage.

## Learning Outcome

By completing this project, you will understand how a manual tester thinks, plans testing, writes test cases, reports bugs, and connects requirements to testing coverage.
